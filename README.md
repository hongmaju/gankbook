# vue02

> A Vue.js project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```


# 运用库
* vue2.0    
* light7
* webpack
* vue-router
